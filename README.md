# Club de Robótica UDP - Sitio Web Oficial

Repositorio oficial de la página web del **Club de Robótica de la Universidad Diego Portales**. Este sitio centraliza la información de los proyectos, documentación y noticias de nuestra comunidad.

## 🚀 Proyectos en Curso

El sitio incluye documentación y detalles técnicos de nuestras iniciativas principales:
- **Brazo Robótico:** Desarrollo de manipuladores controlados.
- **Dron Autónomo:** Investigación en vuelo y navegación programada.
- **Micro Rover:** Vehículos de exploración a pequeña escala.
- **Red de Sensores IoT:** Monitoreo y captura de datos en tiempo real.

## 🛠️ Tecnologías Utilizadas

Este pagina está construido con un stack moderno enfocado en el rendimiento y la experiencia de desarrollo:

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Contenido:** [Velite](https://velite.js.org/) (para gestión de archivos MDX)
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/) y Shaders (GLSL)
